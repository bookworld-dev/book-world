# Book World

## Seed locations

Save all default possible locations (countries & a lot of states/regions) to your local DB:

```
npx tsx scripts/seedLocations.ts
```

## Run locally

```
npm run dev
```

## Run tests

You need to run Docker first, to use test DB containers.

```
npm test
```
